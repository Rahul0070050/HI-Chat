# HI

This is a chat app using react and firebase, It's my side project and also expecting some ideas and your support. Now it's not responsive

### App Features

- [x] login 
- [x] signUp 
- [x] set profile
- [ ] follow and unfollow
- [ ] edit profile
- [ ] change password
- [ ] home
- [ ] about 
- [ ] users info
- [ ] personal chat 
- [ ] group chat
- [ ] block chat
- [ ] report user
- [ ] log out
